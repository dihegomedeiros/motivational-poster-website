# Website de Pôsteres Motivacionais

Um projeto simples desenvolvido para praticar os fundamentos do desenvolvimento web utilizando HTML, CSS e JavaScript.

Este projeto faz parte da minha revisão dos conceitos básicos de Front-end e será evoluído gradualmente, recebendo novas funcionalidades ao longo do tempo.

---

## 📖 Sobre o projeto

O objetivo desta primeira versão é exibir um pôster motivacional composto por:

- Uma imagem
- Um título
- Uma frase motivacional

Nesta versão, o foco está na construção da interface utilizando apenas HTML e CSS.

---

## 🎯 Objetivos de aprendizado

- Estruturação de páginas com HTML5
- Estilização com CSS3
- Organização de arquivos
- Utilização de fontes externas (Google Fonts)
- Manipulação de imagens
- Versionamento com Git
- Publicação no GitHub

---

## 🛠️ Tecnologias

- HTML5
- CSS3
